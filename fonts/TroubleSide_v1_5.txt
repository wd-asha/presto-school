SIL Open Font License 

Бесплатный шрифт для любых форм использования. 
Разработан для серии комиксов Troubleside от Луганской Контемпорари Диаспоры (instagram.com/lkd_dragon).
Вдохновлен шрифтами от Comicraft и классической эстетикой американских комиксов.
Содержит основные символы латинского алфавита и кириллицы (только прописные буквы), буквы с диакритическими знаками, основные лигатуры, цифры, знаки препинания и специальные символы. Вместо строчных букв шрифт содержит альтернативный набор прописных для возможности придания рукописному шрифту большей вариативности.

Поддерживает: английский, русский, украинский, белорусский, французский, испанский, болгарский и другие языки, которые используют схожий алфавит.

V1.5 [26.04.2021]
Добавлено два альтернативных способа ввода кириллических и латинских гласных с ударениями (обычным "´" и обратным "`"):
- через альтернативные глифы.
- через литатуры, для автозамены ввода двух символов "гласная""ударение" без пробела
Пробуйте оба варианта, в зависимости от программ, они могут работать по разному.
EN: Added stylistic alternatives with accents (acute and grave accents) for all Cyrillic and Latin vowels. Also, accented vowels can now be entered through ligatures ("vowel"+"accent" without space).



V1.4 [16.04.2021] 
Поправлено:
- форма восклицательного знака
- увеличен знак $
Добавлено:
- ѝ,Ѝ (за български)
- кириллические ѐ,Ѐ
- кернинг для многоточия
- Скобки "кошачьи усы" ("cat whiskers") для выражения некоторых эмоций в комиксах. Для ввода используйте "{","}"
- Добавлены комбинированные ударения "Combining Acute Accent" и "Combining Grave Accent" с нулевым отступом для быстрого добавления ударений к гласным. Они работают странно, может понадобиться вводить их два раза и из-за фиксированного отступа не удалось настроить кернинг.
- Добавлены денежные символы ¥, ¢, £ 
- inverted punctuation marks "¡", "¿" for Spanish
Благодаря за помощта на Petar Tasev.


V1.3: добавлено 10920 кернинг-пар для всех букв и некоторых знаков препинания. Из-за того, что настройки кернинга строчных букв привязаны к прописным, их ритм может "гулять". Я считаю это допустимым для рукописного шрифта. Если этот момент критичен - используйте только прописные.
